# 트위터 앱

React와 파이어베이스를 통해 만든 트위터 앱입니다.

## 프로젝트 소개

React를 배운 후, 이를 적용해 무언가 만들고 싶다는 생각에 만든 앱입니다.
유튜버 노마드 님의 영상을 참고했습니다.
